# PHP flash messages

Session flash messages for PHP


## Dependencies

* `PHP >= 5.6.0`


## Installation

Install with [Composer](http://getcomposer.org):

    composer require dtkahl/php-flash-message


## Usage

```php
session_start();
$flash = new \Dtkahl\FlashMessages\FlashMessages;
```
